User Defaults

User Defaults хранит данные в plist-файлах. И, как в случае с Property list, данные представляют из себя пары «ключ-значение», примерно так, как вы видели это при работе со словарями

Для работы с User Defaults используется класс UserDefaults, который работает на основе паттерна «Одиночка» (Singleton). Для того, чтобы получить экземпляр класса, необходимо выполнить следующий код: `UserDefaults.standard`

```swift
var userDefaults = UserDefaults.standard
// Запись
userDefaults.set("Some random text", forKey: "Some key")
// Чтение
UserDefaults.standard.object(forKey: "Some key")
```

Метод чтения вернет:

- значение типа Any?, если запрошенный ключ forKey существует;
- nil, если запрошенный ключ forKey не существует.

После того, как метод object вернет значение типа Any?, вам необходимо привести его к типу данных, соответствующему этому значению.
